README for Twenty Fourteen ASU Child Theme
------------------------------------------

1. Install theme as usual.
2. Install "Fourteen Colors" plugin. This is required to customize the colors of the theme.
3. Once theme is installed, activate it and go to Appearance > Customize > Colors and change the Accent Color to #008ED6.
4. First save, and then proceed with any other customizations necessary.

Documented by Max. Last updated on 5/15/14.
